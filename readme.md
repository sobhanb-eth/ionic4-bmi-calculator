## Ionic 4 BMI Calculator App


### About this repo
This repo is the project code of an Ionic 4 app that calculates and classifies the bmi with simple condition flow.
-- This Repo is Archived

### Installation
```
npm install
```

### Browse app
```
ionic serve
```
or

```
ionic lab
```
